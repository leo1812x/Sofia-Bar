# Blue Drank Batch

## Ingredients
- 1L tequila
- 500mL blue curacao
- 250mL passao

## Method
1. Combine all ingredients in a clean container
2. Mix thoroughly
3. Label with name and date
4. Refrigerated

## Storage
- Refrigerated
- Shelf life: 30 days
