# Espresso Martini Batch

## Ingredients
- 3L vodka
- 2L cold brew
- 1L house amaro
- 18.5oz coffee liqueur

## Method
1. Combine all ingredients in a clean container
2. Mix thoroughly
3. Label with name and date
4. Refrigerated

## Storage
- Refrigerated
- Shelf life: 7 days
