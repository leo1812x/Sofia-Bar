# Category 3 Batch

## Ingredients
- 1.5L white rum
- 1.5L dark rum
- 500mL golden falernum

## Method
1. Combine all ingredients in a clean container
2. Mix thoroughly
3. Label with name and date
4. Refrigerated

## Storage
- Refrigerated
- Shelf life: 30 days
